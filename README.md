# 尺码计算器

尺码计算器，是很实用的微信小程序，用于计算衣服、鞋子等尺码的计算器，网购时可以参考。

### 系统说明：

此小程序开发于2017年5月。

支持本程序，请到Gitee和GitHub给我们点Star！

Gitee：https://gitee.com/caozha/wx-chima

GitHub：https://github.com/cao-zha/wx-chima

### 安装方法

直接将源代码复制到您的小程序根目录即可正常使用。
